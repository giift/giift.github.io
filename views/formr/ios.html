FormR-SDK

Usage

To run the example project, clone the repo, and run pod install from the ROOT directory first.

* Swift

Open the FormR.xcworkspace and run the FormUISwift target.

* Objective C

Open the FormR.xcworkspace and run the FormUI target.

Requirements

iOS 7.0 and above

Installation

Add the following line to your Podfile:

platform :ios, '7.0'

pod 'FormR', :git  => 'git@bitbucket.org:giiftalldev/formr-ios-uat.git', :branch => 'master'

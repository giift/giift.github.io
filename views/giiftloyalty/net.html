Installation

Operation of using the .Net.

Firstly, obtain the required files and follow the procedures.

Required Files

  1. App.config
  2. GiiftAPICSharp.dll
  3. Newtonsoft.Json.dll
  4. RestSharp.dll
  5. The executable file(.exe)

Procedures

  1. Place the required files in the same directory.
  2. Edit the App.config file details to suit the program.
  3. Click on the executable file to run the program.

Editing the App.config file

The App.config file will look similar to this format:

<?xml version="1.0" encoding="utf-8" ?>
<configuration>
    <configSections>
        <section
            name="appSettings"
            type="System.Configuration.AppSettingsSection, System.Configuration, Version=2.0.0.0, Culture=neutral, PublicKeyToken=b03f5f7f11d50a3a"
            />
    </configSections>
  <appSettings>
    <add key ="base_url" value ="https://sandbox.giift.com/" />
    <add key="client_id" value ="My Client ID" />
    <add key ="client_secret" value ="My Client Secret" />
    <add key="grant_type" value ="client_credentials" />
    <add key="scope" value ="pos.read pos.write cart" />
  </appSettings>
</configuration>
  1. Replace the phrase “My Client ID” excluding the quotation marks with your relevant Client ID.
  2. Replace the phrase “My Client Secret” excluding the quotation marks with your relevant Client Secret.
  3. Save the file.

Gift Card POS Functionality

Check Balance

To generate the card details.

List of parameters:

Parameters Description
cardNumber The card number that is being registered to the program

Successful response

Upon successful response, its will return the card details of the card with the status and message code.

Status: Success
Message: 200
Card Details : *The generated card details*

Error response

If there is a error, there will be no generation of the card details and the status and message error will be displayed.

Status: Error
Message: *The error message*

Debit Amount from Card

To generate the transaction details.

List of parameters:

Parameters Description
card_id The card id from the card
amt The amount of money to be deducted from the card
extra Additional information if there is any to be included

Successful response

Upon successful response, its will return the transaction details of the card with the status and message code.

Status: Success
Message: 200
Transaction Details : *The generated transaction details*

Error response

If there is a error, there will be no generation of the transaction details and the status and message error will be displayed.

Status: Error
Message: *The error message*

Activate Card

To generate the activation details of the card.

List of parameters:

Parameters Description
dmo_id Program ID
amt The amount of money to be credited into the card
state Optional external id that will be visible in the response. Very useful to reconcile with merchant.
card_number A array of Card Numbers. The card number array cannot be empty
extra Additional information if there is any to be included

Successful response

Upon successful response, its will return the activation details of the card with the status and message code.

Status: Success
Message: 200
Activation Details : *The generated activation details*

Error response

If there is a error, there will be no generation of the activation details and the status and message error will be displayed.

Status: Error
Message: *The error message*

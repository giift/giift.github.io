Overview

The Giift.com API is a REST based API that allows you to access Giift.com services programmatically. In order to get access to the API you need to be a customer of Giift.com and have completed the API onboarding process. Depending on your contract with Giift.com, you will be able to :

  1. Issue gift cards
  2. Top up gift cards
  3. Buy miles/points
  4. Buy Telephone Top Up
  5. Credit points on a loyalty program
  6. Debit value on a loyalty program

Servers

There are two API servers, a production API server, and a test API server, when you are given access to the API from Giift.com staff, you will be given access to both. Before you start using the production API, it is recommended that you use the test API to make sure your application is working correctly. If you need help with your testing, please let us know and we can help you validate that your API code is working correctly.

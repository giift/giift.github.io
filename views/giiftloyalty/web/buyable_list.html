List of cards, products and points available in the API

Retrieve the entire list

The entire list of cards, products, topup of miles and points can all be retrieve in one single call:https://sandbox.giift.com/api/cart/v2/buyable/list.

Filter by country

If you wish to retrieve only the items from one country, just add the parameter in the query country=XX. The country code is using the ISO 3166-1 alpha-2 . Example, if you want to retrieve all items in Giift catalogue in the US, just use:https://sandbox.giift.com/api/cart/v2/buyable/list?country=US.

Filter by type

If you wish to retrieve only the items from one type. These are the current types:

List of parameters:
Type Description
card Gift cards program.
topup All the type of topups. It concerns topup of gift cards (adding value to an existing card), but also points and miles purchases.
product Merchandises and physical products.

Example, if you want to retrieve all gift cards in Giift catalogue, just use:https://sandbox.giift.com/api/cart/v2/buyable/list?type=card.

Display the price in a specific currency

Per default, all the prices are displayed in the currency of the product you are trying to buy. If you want to retrieve the prices in a specific currency, add it to the list of parameter currency=usd in lower case ISO 4217. The field unit will still be displaying the original currency of the object but the file price will display the price in the currency you choose.

Display a specific list

If you are implementing a exchange platform, in a sense where you are in charge of offering a list of redemption possibilities to your users, you might want to restrict the list of cards, topup and products you would like to retrieve. For example, you might want only few carefully chosen US card and few airlines miles program only. In order to retrieve only a sublist, you will be contact with Giift.com Support to set up the desired configuration. Your loyalty program will be added in Giift database and a id will be provided to you. Using this id, you will call: https://sandbox.giift.com/api/cart/v2/buyable/list?buying_with=YOUR_LOYALTY_ID_IN_GIIFT.

You will then only retrieve the list you are entitled to purchase.

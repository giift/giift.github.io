Build your catalog

Whether you already have a redemption catalogue for your customer or you are building one, here are the steps to retrieve all the products from Giift.com and some add-ons to help you filter them.

Complete list

Giift.com API allows you to retrieve the entire list of products available for purchase.

In Giift.com we describe a product by its definition (name, image,Term and Conditions) and the different denominations for which this product can be bought.

Perform a post request to the following url: https://sandbox.giift.com/api/cart/v1/buyable/list> An example of response is shown below.

The call will return a list of product description. The id in this schema is Giift.com internal id called dmo_id sometimes in the rest of the documentation.

This call returns the basic information on the product. We can see that this is a Gift Card (factory_name = card) and the most common values are 10, 25, 50 and 100 USD.

Filter your products per category

Perform the post request to the following url: https://sandbox.giift.com/api/dmo/v1/categorie/all>

To retrieve the details of each category, perdorm the post request on: https://sandbox.giift.com/api/dmo/v1/categories/find>

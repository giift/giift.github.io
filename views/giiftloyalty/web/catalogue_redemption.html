Once you have retrieved your entire list of product, you can display the list on your different applications. The standard procedure is to allow the user to select one product and redirects him/her to another page to display more information about the product.

Perform the post request to the following url to retrieve more details about the product: https://sandbox.giift.com/api/cart/v1/buyable/item/card/{id}> A example of response is shown below.

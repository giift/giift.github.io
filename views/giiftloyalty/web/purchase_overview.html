Purchase API in Giift.com

The Giift.com API is a REST based API that allows you to purchase products available in the Giift.com catalogue. Among them are gift cards to be delivered electronically and products to be mailed.

There is no significant differences in the way Giift handles these two types of products during a purchase.

  1. The user creates a cart object.
  2. The user adds one or several items in his/her cart (gift cards or physical products)
  3. The user converts his/her cart object into an official order.
  4. The user reviews and chooses all the possible method to deliver his/her products (email, sms, mail, simple API response).
  5. The users chooses a payment method and submit a payment.

Nevertheless, the purchase of one Gift Card is a very common one and an express way of doing it is available without going through the previous 5 steps.

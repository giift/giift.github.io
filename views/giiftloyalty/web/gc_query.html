Querying a gift card

This is the same as what can be found on the loyalty program.

The idea behind this call is to retrieve the necessary information on a customer and its loyalty program card.

Natural subsequent calls are to redeem or to add points to the card.

Perform a GET request to the following url: https://sandbox.giift.com/api/card/v1/lookup

This method will return the list of cards matching the query. Unless the query is a card number, the result can be a mix of gift cards and loyalty programs to whom the current user has access.

Only one string parameter. It can be chosen among the following list:
Parameter Description
q This is a very flexible but slower way of querrying for a loyalty program. The system, will try to solve the query in the following order: card_number, email, phone.
card_number The loyalty card number, This is usually a 16 digits number but it can also contain letters and special caracters.
email The user's email.
phone The user's telephone number.

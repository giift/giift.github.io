Topup a gift card

There are 2 ways to topup a gift card. The express allows adding value to a card by a single call. The advanced method is more complicated, but it gives a fine control on every step of the process of adding value to a card.

Express

This method only works if you have a retailer account which allows post paid billing or you have a prepaid account. For on the fly payments, using the dongle, paypal or similar payment methods, you need to use the advanced method.

If you don't have access to it, but you are interested, please contact your account manager.

Perform a POST request to the card/credit/<card_id> method.

Parameters are as follow:

Parameter Description
amt Amount in the card currency to be credited.

Advanced

Using the cart api, it is possible to control every step of the topup process. It is not fully documented yet, but if you need it urgently, please contact us.

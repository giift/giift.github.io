Give Loyalty points

There are 2 ways to add points to a loyalty program. One way is to directly specify a number of points to be added to the card. The other way, is to specify the underlying financial transaction and the system will automatically convert it into a number of points using the exchange rate setup in the system.

Points

Perform a POST request to the card/credit/<card_id> method.

Here is the list of parameters:

Parameter Description
card_id The gift card id.
amt Amount in the program unit to be credited.

Cash

Perform a POST request to the card/credit/<card_id> method.

Here is the list of parameters:

Parameter Description
card_id The gift card id.
amt Amount in cash of the underlying transaction. This will be converted to points before being added to the card.
type Should be cash.
exchange_id Id of the exchange that will convert a financial value into points. All exchange ids can be retrieved using the method exchange/around.
